# Project 2 - Giphy Search Engine
**Cassandra Aguero
## Overview
This project is a one-page website that allows users to search for GIFs using the Giphy API.
Users enter a keyword, submit the form, and the site displays matching GIF results in a responsive grid.

## How to Use
1. Type a keyword here
2. Click **Search**.
3. Scroll through the GIF results displayed below the search area.

## Technologies Used
- HTML
- CSS
- JavaScript
- Giphy API
